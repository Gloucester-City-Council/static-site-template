# Static Site Template (Council Pilot)

This repository is a **template** for public, static websites in the council GitHub pilot.

## Governance

- [Security Policy (local copy)](SECURITY.md)
- [Data Classification (local copy)](DATA-CLASSIFICATION.md)

**Master versions:**
- [Security Policy – master](https://github.com/Gloucester-City-Council/github-standards/blob/main/SECURITY.md)
- [Data Classification – master](https://github.com/Gloucester-City-Council/github-standards/blob/main/DATA-CLASSIFICATION.md)

## How to use

1. Click **Use this template → Create a new repository**.
2. Name the new repository after the site, e.g. `waste-services-site`.
3. The new repo will copy this structure.
4. Make changes via pull requests.

## Scope

- Public-facing, static HTML/CSS/JS only.
- No personal data.
- No secrets, API keys or internal system details.

See the `github-standards` repository for full security and data rules.
